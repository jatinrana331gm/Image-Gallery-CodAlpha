# 🌟 Image Gallery  

A dynamic and responsive image gallery built with **HTML, CSS, and JavaScript**, featuring **category filtering, search functionality, dark mode, interactive previews, fullscreen view, and direct image downloads**.  

---

## 🌐 Live Demo
[![Live Demo](https://img.shields.io/badge/Live%20Demo-Glow%20Gallery-brightgreen?style=for-the-badge)](https://jatinrana331gm.github.io/Image-Gallery-CodAlpha)

---

## 🚀 Features  

✅ **Categorized Image Filtering** - Filter images by "Place & Travel," "Animals," "Art & Creativity," and "Food & Drinks."  
✅ **Search Functionality** - Quickly find images by typing keywords.  
✅ **Dark Mode** - Toggle between light and dark themes.  
✅ **Interactive Image Previews** - Click to view a larger preview in a lightbox with navigation buttons.  
✅ **Fullscreen View** - View images in fullscreen for immersive experience.  
✅ **Direct Image Download** - Download images directly from the preview lightbox.  
✅ **Responsive Design** - Adaptable to different screen sizes.  
✅ **Smooth Transitions & Animations** - Enhances user experience.  

---

## 🛠️ Technologies Used  

- **HTML** - Structure of the webpage  
- **CSS** - Styling and responsive layout  
- **JavaScript** - Interactivity, filtering, search, dark mode, lightbox  

---

## 📌 How to Use  

1. Open the `index.html` file in your web browser.  
2. Use **category filters** to view specific images.  
3. Use the **search bar** to find images quickly.  
4. Click an image to open the **preview lightbox**.  
5. Use navigation (**previous/next**) buttons in the preview.  
6. Click **fullscreen** for full immersion.  
7. Click **download** to save images.  
8. Use the **Dark Mode toggle** to switch themes.  
9. Click **'X'** to close the preview window.
